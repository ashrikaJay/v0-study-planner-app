# v0-study-planner-app

This is a [Next.js](https://nextjs.org) project bootstrapped with [v0](https://v0.app). StudyPath is an AI-powered learning roadmap generator and resource organizer built with Next.js, v0, and the Vercel AI SDK. Users enter any topic and the app generates a structured zero-to-expert learning roadmap across four stages — Beginner, Intermediate, Advanced, and Expert. Resources such as articles, documents, or notes can be pasted in and automatically classified into the appropriate stage with a priority tag (Read First, Read Later, or Optional). A personal dashboard tracks all classified resources with progress checkboxes, helping learners stay organized on their journey from complete beginner to expert on any subject.
